Graph Reduction

This page uses a tiny S-expression surface syntax restricted to binary pairs:

  • () is the empty list
  • (a b) is a pair (also used as an application node)

The core reducer is split into two small steps:

  • resolve instantiates motifs expressed with non-negative integer indices. Indices are in fill order: in (((f x) y) z), the environment is [x, y, z], so 0 -> x, 1 -> y, 2 -> z, etc.
  • collapse performs one leftmost-outermost step of the single rewrite: (() x) -> x

Example motif (S kernel body):

((0 2) (1 2))

Applied as:

(((((0 2) (1 2)) a) b) c)

resolve reduces that in one step to:

((a c) (b c))
